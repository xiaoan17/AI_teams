# AI Teams Logo Exploration

Brand direction inferred from the product:

- local-first multi-agent terminal workspace
- chat-style routing, explicit `@` dispatch, and terminal truth
- docs and Git as durable context
- restrained workbench UI with cold gray surfaces and amber focus

The logo set keeps the same discipline: monochrome SVGs using `currentColor`, generous negative space, rounded cuts, and compact geometry that survives app-icon and sidebar sizes.

## Variants

| File | Direction | Rationale |
| --- | --- | --- |
| `svg/01-router-glyph.svg` | Router Glyph | Three agent inputs converge into one command rail. Strongest match for `@all` and message routing. |
| `svg/02-team-orbit.svg` | Team Orbit | Multiple agents move around a human command center. Softer, more product-brand friendly. |
| `svg/03-handoff-stack.svg` | Handoff Stack | Task documents become the carrier for work. Best for the Docs + Git as context promise. |
| `svg/04-terminal-relay.svg` | Terminal Relay | Three terminal panes connected by orchestration. Most literal for the desktop app. |
| `svg/05-worktree-fork.svg` | Worktree Fork | Git worktrees and parallel branches. Strong engineering signal. |
| `svg/06-doc-matrix.svg` | Doc Matrix | Structured files, pinned facts, and one routed output. Compact, icon-friendly. |
| `svg/07-focus-pulse.svg` | Focus Pulse | Status, waiting input, and focused attention. Best fit for the current ambient effects language. |
| `svg/08-local-mesh.svg` | Local Mesh | A private orchestration network inside a bounded local container. Broadest platform metaphor. |

## Color Use

The SVGs are intentionally single-color. Recommended app colors:

- Workbench foreground: `#e8edf2`
- Amber brand accent: `#f2c14e`
- Focus blue: `#5f83b6`
- Dark icon plate: `#10151a`

## Next Selection

For the current product taste, the strongest shortlist is:

1. `01-router-glyph`
2. `03-handoff-stack`
3. `07-focus-pulse`

Pick one or two and refine before replacing the current `AT` mark in the app.
