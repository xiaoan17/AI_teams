# AI Teams Light Logo Direction

This pass intentionally uses a bright identity system instead of the dark workbench mood.

Brand angle:

- not "AI brain" and not "team people"
- product primitive: command dispatch layer
- visual metaphor: a precise light control surface where one human command routes work to multiple local agents

Palette:

- porcelain white: `#f8faf7`
- mist gray: `#e8edf2`
- ink: `#17212b`
- route blue: `#4f8edb`
- cursor amber: `#f2b84b`
- calm green: `#4fbf93`

Recommended shortlist:

1. `01-light-dispatch-core` - strongest product primitive, best app icon candidate.
2. `03-paper-router` - best if Docs + Git handoff should be the brand promise.
3. `04-agent-compass` - best if the brand should feel like supervision / situational awareness.

The app-icon PNGs are preview renderings. The source SVGs remain one-color `currentColor` symbols for implementation.
